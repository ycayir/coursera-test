Bootstrap Based Web Page Example

Steps to produce this page:

  1. Download the Bootstrap from http://getbootstrap.com/docs/4.0/getting-started/download/
  2. Put the css and js folders into lecture25 folder.
  3. Download jQuery from http://jquery.com/download/
  4. Put jquery-3.2.1.min.js into js directory
  5. Download Popper from https://popper.js.org
  6. Put popper.js folder into lecture25 directory
  7. And lastly create this page with all dependencies.
